# myblog

一个Flask框架搭建的简易博客
项目演示地址：https://wakingup.herokuapp.com/

<br>

---
#### 整体架构
- 整体：Flask+SQLAlchemy
- 前端:Bootstrap+CSS+flask-WTF
- 文章编辑器：Flask-SimpleMDE
- 后台管理：Flask-Admin


---
#### 功能
支持用户登录/文章/说说/标签/评论/友链/api等

<br>
